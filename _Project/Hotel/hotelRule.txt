호텔의 데스크직원이 사용할 수 있는 호텔 객실 관리프로그램

Java 어플리케이션 또는 웹브라우저를 실행가능한 개인PC에서 사용합니다.
사용자는 제공하는 기능 메뉴를 보고 입력을 통해 기능을 활용할 수 있습니다.

1. 현재의 객실의 상태와 집계된 개수를 확인 가능합니다.
- 빈 객실
- 예약된 객실
- 투숙중인 객실

2. 예약된 또는 투숙 중인 객실의
   고객명과 기본정보(전화번호 | 생년월일)를 열람해 확인 가능합니다.

3. 객실 상태 변경 기능
- 예약  |  예약취소  |  예약변경
- 체크인  |  체크아웃

4. 객실 및 고객 조회 기능
- 방번호로 검색 | 고객명으로 검색 | 전화번호로 검색

--------------------------------------

A동

2층부터 5층까지 일반 등급의 객실이 존재. 각 층마다 20개의 객실이 있다.
방번호 규칙 : 3자리의 정수. 첫자리는 층수를 의미한다.
방 분류
싱글 베드 : 방번호의 끝자리 수는 홀수이다.
더블 베드 : 방번호의 끝자리 수는 짝수이다.
